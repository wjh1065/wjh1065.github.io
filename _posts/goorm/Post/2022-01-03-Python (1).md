- Immutable Types & Mutable Types
파이썬에서 대입은 원칙적으로 메모리 주소 복사. 
(즉, 값을 복사하지 않고 같은 주소를 공유함)

불변형의 경우 수정이 필요한 경우에 *새로운 객체* 를 생성.

```python
>>> a = 1  
>>> b = 1 
>>> a is b
True

>>> a = 13453436
>>> b = 13453436
>>> a is b
False

>>> a = ‘text’  
>>> b = ‘long-long-text’  
>>> a is ‘text’, a == ‘text’, b is ‘long-long-text’, b == ‘long-long-text’
(True, True, False, True) 

>>> a = True
>>> a is True
True

>>> a = None
>>> a is None
True
```
